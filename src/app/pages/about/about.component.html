About Us

This is a simple landing page application built with Angular 19 and styled with Tailwind CSS v3.

The application demonstrates:

  • Angular 19 standalone components
  • Client-side rendering (no SSR)
  • Tailwind CSS v3 for styling
  • Routing between pages
  • Responsive design

Our Mission

We're here to showcase how easy it is to build modern, beautiful web applications using the latest Angular framework combined with the utility-first CSS framework, Tailwind CSS.
